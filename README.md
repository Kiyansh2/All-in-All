# All in All V2 — Option C Separate Premium

This is the simplified no-folder version.

Upload these files directly to the root of your GitHub Pages repository:

- index.html
- style.css
- script.js
- favicon.png
- social-preview.svg
- All-in-All-Extension.zip
- README.md

Do not unzip `All-in-All-Extension.zip`. It is the downloadable browser extension file used by the website.

This version intentionally separates itself from the main All in All Platform design while preserving a premium ecosystem vibe. It includes:

- Center-aligned layout
- Number count-up animations
- Clean quick launch links
- Product demo previews
- Extension download section
